Using

Download the hex file from the releases.

—OR—

This repository can be added as an extension in MakeCode.

  • open https://makecode.microbit.org/
  • click on New Project
  • click on Extensions under the gearwheel menu
  • search for https://github.com/chessgeekboy/flappy-bird and import

Edit this project

Info for editing outside of makecode editor coming soon.

To edit this repository in MakeCode.

  • open https://makecode.microbit.org/
  • click on Import then click on Import URL
  • paste https://github.com/chessgeekboy/flappy-bird and click import

To edit this project in VSCODE.

  • Clone this repo into a directory.
  • Open vscode in that directory.
  • Run the following commands in a terminal opened in that directory.
  • Download pxt with npm install -g pxt.
  • Target the microbit with pxt target microbit.
  • Install with pxt install.
  • Build with pxt build and the hex file with be in the build folder called "binary.hex".
